[Dileep_Wipro_Resume.pdf](https://github.com/user-attachments/files/23809381/Dileep_Wipro_Resume.pdf)#  E-Commerce Application for Food Sale

A full-stack web application designed for browsing food items, adding products to cart, placing orders, and managing product data. Built with **Java, Spring Framework, HTML, CSS, Bootstrap, and MySQL**.

---

##  Project Overview

This project is a dynamic and responsive **E-Commerce Food Ordering Platform** with complete functionality from product listing to order placement.  
It demonstrates strong knowledge in backend development, MVC architecture, database handling, and user-friendly UI design.

---

##  Tech Stack

### **Frontend**
- HTML  
- CSS  
- Bootstrap  

### **Backend**
- Core Java  
- Spring Framework  
- Spring MVC  

### **Database**
- MySQL  

---

##  Features

-  Browse food products  
-  Add to Cart  
-  Place orders  
-  User login & session management  
-  Admin product CRUD  
-  Spring MVC routing  
-  MySQL database integration
